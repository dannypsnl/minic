# minic

A course level compiler to program in Lean, mainly reference to Indiana University courses[^1]

## Usage

Before you start, you will need to [install lean prover](https://leanprover.github.io/lean4/doc/setup.html). Once you done, you can run the following makefile command to run examples in this project.

```shell
make run
```

[^1]: https://iucompilercourse.github.io/IU-P423-P523-E313-E513-Fall-2020/
